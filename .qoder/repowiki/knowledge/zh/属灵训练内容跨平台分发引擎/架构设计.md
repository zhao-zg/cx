- **中心化构建编排**：`main.py` 作为核心控制器，解析 `resource/` 原始文档并产出标准化的 `training.json` 及静态资产至 `output/`，确立单一数据源。
- **共享数据契约**：Web SPA 与 Android Client 均消费同一套由生成器产出的 JSON 结构与静态资源，确保跨平台内容一致性。
- **混合应用桥接**：利用 `capacitor.config.json` 将 `output/` 映射为 Android 原生壳的 Web 根路径，实现 PWA 站点在移动端的无缝加载。
- **配置注入机制**：全局配置（如 API 地址）在构建阶段经 Base64 混淆后编译为 `remote-config.js`，自动同步至所有前端运行时环境。
