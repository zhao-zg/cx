- **统一构建管线**：`main.py` 作为核心编排器，解析 `resource/` 目录下的 Word/TXT 源文件，生成标准化的 `training.json` 及静态资源至 `output/` 目录。
- **共享数据契约**：所有子模块（Web SPA、Android Client）均消费由生成器产出的同一套 JSON 数据结构与静态资产（HTML/CSS/JS），确保内容一致性。
- **混合应用桥接**：通过 `capacitor.config.json` 将 `output/` 目录定义为 Web 资源根路径，使 Android 原生壳能够直接加载并渲染由 Python 生成的 PWA 站点。
- **配置同步机制**：`config.yaml` 中的远程服务器地址等全局配置在构建时被编译为 `remote-config.js`，自动注入到 Web 前端与 Android 应用的运行时环境中。
