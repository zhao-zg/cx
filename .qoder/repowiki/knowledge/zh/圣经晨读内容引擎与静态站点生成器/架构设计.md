- **后端构建层 (Python)**: 采用 `ImprovedParser` 解析 `.doc/.docx` 源文件，提取纲目、听抄、晨兴喂养等结构化数据至 `models.py` 定义的 Dataclass；通过 `HTMLGenerator` 利用 Jinja2 模板引擎生成静态资源（JSON/JS/CSS），并维护增量式经文字典 (`bible_dict.py`)。
- **前端运行时 (SPA/PWA)**: 基于 `index.html` 入口，采用 Hash 路由 (`router.js`) 驱动 `renderer.js` 进行客户端 JSON 到 DOM 的动态渲染；集成 Service Worker (`main_sw.js`) 实现缓存优先策略，确保离线可用性。
- **数据流边界**: 构建阶段产出 `training.json` 及 `search-index.json`；运行阶段通过 `localforage` 管理本地导入训练，实现动态内容扩展。
