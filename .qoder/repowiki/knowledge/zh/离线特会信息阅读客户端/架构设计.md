- **混合架构**：采用 Capacitor 6 桥接原生 Android 与 Web 前端。`MainActivity` 继承 `BridgeActivity`，在 `onCreate` 中注册自定义插件并加载 `assets/public/index.html`。
- **原生层 (Java)**：位于 `com.tehui.offline` 包。包含 `NativeTTSPlugin`（通过 `TTSForegroundService` 实现后台语音合成）、`ApkInstallerPlugin`（应用内更新）及崩溃日志收集器 `CrashReporter`。
- **Web 层 (JS)**：位于 `assets/public/js`。核心由 `router.js`（Hash 路由）、`renderer.js`（JSON 数据渲染）和 `speech.js`（双引擎 TTS 适配）组成。支持 Service Worker (`sw.js`) 实现 PWA 离线缓存与资源包动态加载。
- **数据流**：静态 JSON 数据（`training.json`）存储在 `assets` 中，通过 `localforage` 和 Cache API 进行本地持久化与版本管理。
