- 原生插件遵循 Capacitor `@CapacitorPlugin` 注解规范，通过 `PluginCall` 处理 JS 与 Java 间的异步通信。
- 前端采用 Hash 路由模式（`#/path/view`），通过 `CXRouter` 统一管理页面导航与历史记录栈。
- 视图渲染采用“数据驱动”模式，`renderer.js` 根据 `training.json` 结构动态生成 DOM，并注入经文引用高亮与上下文扫描逻辑。
- 语音合成模块 `speech.js` 实现了 NativeTTS 与 Web Speech API 的双引擎自动降级，并通过 `MediaSession` 集成系统锁屏控制。
