# 朗读功能说明

## 功能概述
本站使用浏览器内置的 Web Speech API 提供文本朗读功能，无需额外安装任何插件或应用。

## 离线朗读限制

### 桌面端
- **Chrome/Edge（桌面版）**：✅ 支持离线朗读（使用系统内置语音）
- **Firefox**：⚠️ 部分支持（取决于操作系统）
- **Safari（macOS）**：✅ 支持离线朗读

### 移动端
- **iOS Safari**：❌ **需要网络连接**
  - iOS 的语音合成服务需要联网才能工作
  - 这是 iOS 系统的限制，不是本站的问题
  
- **Android Chrome**：⚠️ **部分支持**
  - 如果设备已下载中文语音包，可以离线使用
  - 否则需要网络连接
  
- **Android 其他浏览器**：⚠️ 取决于浏览器和系统设置

## 解决方案

### 方案 1：使用桌面浏览器（推荐）
在电脑上使用 Chrome、Edge 或 Safari 浏览器，可以完全离线朗读。

### 方案 2：Android 下载语音包
1. 打开手机设置
2. 进入"语言和输入法" → "文字转语音输出"
3. 下载中文语音包
4. 下载完成后即可离线使用

### 方案 3：保持网络连接
在移动端使用时，保持 WiFi 或移动数据连接。

## 错误提示说明

- **"离线模式"**：检测到设备离线，但仍会尝试播放
- **"需要网络"**：语音服务需要网络连接
- **"语音不可用"**：浏览器不支持或语音服务未启用
- **"播放失败"**：其他播放错误

## 技术说明

Web Speech API 是浏览器标准 API，其离线能力取决于：
1. 浏览器实现
2. 操作系统支持
3. 是否已下载语音包

这不是本站可以控制的，而是由浏览器和操作系统决定的。

## 建议

如果您经常需要离线使用朗读功能，建议：
1. 使用桌面浏览器（Chrome/Edge/Safari）
2. 或在 Android 设备上预先下载中文语音包
3. iOS 用户需要保持网络连接
